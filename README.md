# AKRAV Ai - نظام أمني متكامل

## الملفات المطلوبة:
1. `auth.js` - حارس أمني مركزي
2. `scripts.js` - دوال مساعدة مشتركة
3. `update-index.js` - تضاف لـ index.html
4. `update-ai.html` - تضاف لـ ai.html
5. `update-img.html` - تضاف لـ img.html
6. `ht.html` - صفحة جديدة لمعاينة HTML

## خطوات التثبيت:

### الخطوة 1: إنشاء الملفات
```bash
touch auth.js scripts.js ht.html
